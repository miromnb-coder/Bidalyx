# Bidalyx

Bidalyx is an Expo + React Native + TypeScript mobile app concept for AI-assisted quote creation and quote management for service businesses.

## Development

Install packages:

npm install

Start with Expo Go tunnel:

npm run start:go

Start with cleared cache:

npm run start:clear

Check the project:

npm run doctor
npm run typecheck

## VS Code tasks

The project includes .vscode/tasks.json, so in Codeanywhere / VS Code you can run these tasks:

- Expo Go - Start tunnel
- Expo Go - Start tunnel + clear cache
- Open web preview
- Check Expo project
- Fix Expo dependencies
- TypeScript check
- Install packages
- Reinstall packages
- Quick repair

## Current foundation

This first foundation includes Expo Router navigation, bottom tabs, Bidalyx light theme, reusable UI components, mock quotes and customers, dashboard screen, quote list screen, create quote screen, AI draft preview screen, and customer quote preview screen.
