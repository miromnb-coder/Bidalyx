# Bidalyx

Bidalyx is an Expo + React Native + TypeScript mobile app for quote creation and quote management for service businesses.

## Development

Install packages:

npm install

Start with Expo Go tunnel:

npm run start:go

Start with cleared cache:

npm run start:clear

Check the project:

npm run doctor
npm run typecheck

## Feature roadmap in progress

- document preview
- reusable templates
- message templates
- CRM improvements
- analytics
- reminders
- public quote page
- service specific forms
- SaaS plan foundation

## Current foundation

The app includes Expo Router navigation, bottom tabs, Supabase auth, company setup, quote management, image upload and public quote links.
